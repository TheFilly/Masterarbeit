# Research Notes

Store phase-specific research artifacts here.

- `summary.md`: consolidated state of a phase.
- `open-questions.md`: unresolved issues that can affect later work.
- `findings/`: atomic observations with evidence and impact.
- `format-matrix.csv`: structured inventories or matrices.
- `evidence/`: references and traceability notes.
